# ⚡ React UI Kit

A small React + TypeScript UI component library I am building to practice creating reusable components, Storybook documentation, and Tailwind CSS styling.

## 🧰 Tech Stack

- ⚛️ React
- 📘 TypeScript
- ⚡ Vite
- 📖 Storybook
- 🎨 Tailwind CSS

## 🧩 Components

- `Button` – primary / secondary variants
- `Input` – optional label and error message
- `Modal` – basic dialog overlay
- `Card` – simple container with optional title
- `Checkbox` – labeled checkbox for forms
- `Badge` – small status label (neutral / success / warning / danger)

## 🚀 Development

```bash
# install dependencies
npm install

# run example app
npm run dev

# open Storybook component playground
npm run storybook
